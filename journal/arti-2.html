Fixing some "serious" tech issues

Yesterday my Mum gave me a second hand desktop computer from her office, which is a Lenovo ThinkCentre, originally that desktop has wiped all the things from the hard drive, so I reinstall the Windows 10 into the desktop. After I installed the OS into the desktop, there are some "serious" tech issues, including I cannot connect to the internet and the graphics are awful.

So I tried to install the drivers from the internet using my MacBook Air which is running Windows 10 into my USB Drive, then I plug in to my ThinkCentre desktop, after the installation, the WIFI selection didn't show any network. Luckily, the desktop can be connect to internet using Ethernet cable, so I plug the Ethernet cable into my desktop, after that, it connects to the internet.

Then I installed the graphic drivers from Lenovo's website, after that, the screen's resolution was become normal and clear!

I can say that it is very common when you got a second hand computer.
